# Manual de Governança da Assessoria de Parcerias e Programas Integrados (Maual.APPI)
O Manual.APPI é um guia prático para servidores do Instituto Estadual do Patrimônio Histórico e Artístico de Minas Gerais (IEPHA-MG) que atuam na celebração e gestão de colaborações institucionais.
